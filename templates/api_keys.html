Example Usage

Once you create an API key, use it like this:

curl -X POST https://yourdomain.com/api/generate \
  -H "Authorization: Bearer qb_xxxxx:your_secret_key" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "A brave knight with silver armor",
    "generation_type": "character",
    "quality": "medium",
    "size": "1024x1024"
  }'

Important: Keep your API key secret! Don't commit it to version control or share it publicly.
