What Is Quickbrush?

Quickbrush is a Bring Your Own Key (BYOK) tool that lets you generate AI-powered fantasy RPG artwork directly in Foundry VTT using your own OpenAI API key. Whether you're a Dungeon Master crafting a scene, a player dreaming up your next character, or simply someone who loves the wonder of creation— Quickbrush is here to help you make something beautiful.

How Does It Work?

Using Your Own OpenAI API Key

Quickbrush is a BYOK (Bring Your Own Key) tool. You'll use your own OpenAI API key, giving you full control over costs and usage. The module supports both GPT-Image-1-Mini (faster, cheaper) and GPT-Image-1 (higher quality) models.

Simply install the Foundry VTT module, configure your OpenAI API key in the settings, and start creating! Describe your vision, pick your quality level and model, and let Quickbrush do the rest. In moments, you'll have something unique—art that feels like it came right out of your imagination.
